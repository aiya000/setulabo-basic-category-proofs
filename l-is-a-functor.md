# 関手`L : Sets -> Tord`が関手の定義を満たすこと。

```
Setsの任意の対象について、恒等射は明らかに保存されている。

L(f ○ f^-1)
    = L(id_{x}) ―― 可換性による
    = ≦11
    = ≦11 ○ ≦11 ―― 任意の射は恒等射を合成しても等しい（圏の公理）
    = L(f) ○ L(f^-1)

L(g ○ f)
    = L(h)
    = ≦13
    = ≦13 ○ ≦11 ―― 任意の射は恒等射を合成しても等しい
    = L(g) ○ L(f)

L(h ○ f^-1)
    = L(g) ―― 可換性による
    = ≦13
    = ≦13 ○ ≦ 11 ―― 任意の射は恒等射を合成しても等しい
    = L(h) ○ L(f^-1)
```
